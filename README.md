# Landing Page Himpunan Mahasiswa Informatika

Nama: Khaera Mir'ah Andina

## Deskripsi
Website landing page untuk Himpunan Mahasiswa Informatika Kabinet Ciptakara.
Dibuat menggunakan Laravel dengan fitur routing, blade template, dan configuration.

## Fitur
- Menampilkan nama kabinet
- Menampilkan visi dan misi
- Menampilkan tagline
- Menampilkan kontak email

## Cara Menjalankan
1. composer install
2. php artisan serve
3. buka http://127.0.0.1:8000

## Teknologi
- Laravel
- PHP
- Blade Template
